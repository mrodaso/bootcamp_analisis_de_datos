# Proyecto de Visualización

Este proyecto contiene scripts para visualizar datos usando Python.

## Estructura

- `main.py`: Script principal para ejecutar la visualización.
- `src/visualization.py`: Módulo con funciones de visualización.
- `src/code.py`: Módulo con datos y funciones auxiliares.

## Ejecución

Para ejecutar el proyecto, usa:

```bash
python main.py
```

Asegúrate de tener Python 3.12 o superior instalado.

## Requisitos

Instala las dependencias necesarias (si las hay) usando:

```bash
pip install -r requirements.txt
```
